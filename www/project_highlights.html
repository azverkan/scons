21 May 2006: Google's Summer of Code will fund a project proposed by Philipp Scholl to add support for packaging and release dependencies to SCons. Stefan Seefeld will mentor.

21 May 2006: Tracking of SCons bug reports, patches and feature requests has been moved from our SourceForge project page to our Issue Tracker here at tigris.org. Many thanks to Greg Noel and Kevin Maples for making this happen.

27 April 2006: SCons has been accepted as a Google Summer of Code project. Check out our list of project ideas over at the SCons wiki.

18 August 2004: Beta release 0.96 adds Fortran 90/95 support, better Qt support, platform-independent file manipulation actions, new debugging features, and lots more.
