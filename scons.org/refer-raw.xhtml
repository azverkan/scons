DIANE
DIANE is a lightweight distributed framework for parallel scientific applications in master-worker model. Some of the applications include: Simulation for Radiotherapy and Astrophysics, BLAST for Genome Sequencing, Data Analysis for High Energy Physics and Image Rendering. It is a thin software layer which easily works on top of more fundamental middleware such as LSF, PBS or the Grid Resource Brokers. It may also work in a standalone mode and does not require any complex underlying software. Applications may be run in local clusters or on the Grid without any special modifications. DIANE is using SCons primarily to build extension modules for Python.
